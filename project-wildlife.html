Academic Project: Design Science Research for Wildlife Conservation.

Academic Project
Researcher & Designer
Academic / DSR
Design Science Research User Personas Journey Mapping Prototype Testing Figma

Wildlife Conservation App

Goal: Promote wildlife education & awareness

This project utilized Design Science Research (DSR) methodology to bridge the gap between theoretical insights and practical solutions. The goal was to design a mobile platform that actively promotes wildlife education and conservation awareness.

The Problem

The core research question was: How can digital tools increase engagement in conservation efforts? We needed to move beyond passive information and create an experience that motivates users to care about conservation.

Research & Ideation

I conducted surveys and semi-structured interviews to clearly define user motivations. By analyzing this data, we created detailed User Personas and Journey Maps to visualize the ideal user experience.

Design Iterations

Using Figma, I developed a high-fidelity prototype. We then conducted rigorous usability testing to refine the navigation structure based on real user interactions.

Key Outcomes

  • Identified Content Clarity as the key driver of engagement.
  • Significantly improved layout and accessibility features.
  • Validated theoretical insights through practical prototype testing.
  • Created a scalable framework for conservation education apps.
